MVP (Minimum Viable Product)

Authentication & Account Management

-[] US-1: Register Account
As a new user, I want to register using AWS Cognito so that my backlog is private and securely stored.

US-2: Login
As a returning user, I want to log in securely so that I can access my personal media backlog.

US-3: Session Persistence
As an authenticated user, I want my session to persist while I am active so that I do not need to repeatedly enter my credentials.

US-4 : Logout
As a logged-in user I want to log out of my account. So that my session ends and no one else can access my backlog from my device.

Backlog Management

US-5: Search Media
As a user, I want to search for movies and TV shows using the TMDB API so that I can add real media items to my backlog.

US-6: Add Media to Backlog
As a user, I want to add a movie or TV show to my backlog so that I can track what I plan to watch.

US-7: Set Media Status
As a user, I want to assign a status (Planned, In Progress, Completed, Dropped) to each item so that I can organize my media.

US-8: Update Media Status
As a user, I want to update the status of a backlog item so that my progress stays accurate.

US-9: Remove Item from Backlog
As a user, I want to remove an item from my backlog so that I can keep my list current.

US-10: Able to view all backlog items
As a user, I want to view all my backlog items in one place so that I can easily see everything I am tracking.

Notes, Ratings, and Tracking

US-11: add/edit notes
As a user, I want to add personal notes to a backlog item so that I can record my thoughts and reactions.

US-12: Rate Media Items
As a user, I want to rate completed media so that I can reflect on how much I enjoyed it.

US-13: See date the backlog items are added 
As a user, I want the system to store the date I added an item so that I can see how long it has been in my backlog.

US-14: see date I completed backlog item
As a user, I want the system to store the date I completed an item so that I can track my viewing history.

Dashboard & User Interface

US-15: View Dashboard Summary
As a user, I want a dashboard that displays counts of my media by status so that I can quickly understand my progress.

US-16: Filter Backlog
As a user, I want to filter my backlog by status, rating, or title so that I can find items easily.

US-17: Responsive Interface
As a user, I want a responsive interface so that I can use the system on desktop or mobile devices.

Stretch Goals (Future Enhancements)

US-18: Add Video games
As a user, I want to add video games to my backlog so that I can track all media types in one system.

US-19: Track Episode Progress
As a user, I want to track TV episodes individually so that I can monitor detailed progress.

US-20: Media Recommendations
As a user, I want the system to recommend new media so that I can discover content based on my interests.

US-21: View completion trends
As a user, I want to view charts of my completion trends so that I can analyze my media habits over time.

US-22: Share Backlog or Reviews
As a user, I want to share my backlog or reviews so that I can connect with others.
