Authentication / Account
As a new user, I want to register for an account using AWS Cognito so that my backlog is private and persistent. As a returning user, I want to log in securely so that I can access my personal media backlog.
As an authenticated user, I want to remain logged in during my session so I don’t have to re-enter credentials repeatedly.


Backlog Management (Core Feature)

As a user, I want to search for movies and TV shows via TMDB so that I can add real media items to my backlog.
As a user, I want to add a movie or TV show to my backlog so that I can track what I plan to watch.
As a user, I want to assign a status to each backlog item (Planned, In Progress, Completed, Dropped) so that I can organize my media.
As a user, I want to update the status of a backlog item as I progress through it. 
As a user, I want to remove an item from my backlog if I no longer care about it.
As a user, I want to view all items in my backlog in one place.

Notes, Ratings, and Tracking

As a user, I want to add personal notes to a backlog item so I can record my thoughts. 
As a user, I want to give a rating (1–5 or 1–10) to completed media.
As a user, I want the system to store the date I added an item to my backlog.
As a user, I want the system to store the date I marked an item as completed.

Dashboard & UI

As a user, I want a dashboard that shows counts of my media by status (Planned, In Progress, Completed, Dropped).
As a user, I want to filter my backlog (by status, rating, or title).
As a user, I want a responsive web interface so I can use the system on desktop or mobile.
